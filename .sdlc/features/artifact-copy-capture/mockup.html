Specification: Artifact and Dialogue Copy & Screenshot

Status: Approved · v1.0 · 2026-03-05

Problem

The sdlc UI renders artifacts and ponder dialogues beautifully but provides no path to extract that content for sharing. Users resort to OS-level screenshot keybindings and manual text selection.

User Stories

  • Copy raw markdown from any artifact in one click
  • Copy rendered artifact as image for Slack / Discord
  • Copy individual dialogue messages from ponder sessions

Technical Approach

Copy MD → navigator.clipboard.writeText(artifact.content)
Copy IMG → html2canvas(el) → ClipboardItem({ 'image/png': blob })
Download → html2canvas(el) → canvas.toDataURL() → anchor.click()

Click [MD] or [IMG] to see the feedback state animation.

Ponder session — hover a message bubble to reveal the copy icon. Toggle mobile mode to see always-visible variant.
