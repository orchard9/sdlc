"How does Fix Right Away diagnose issues?"
FixRightAwayModal.tsx:84 routes/diagnose.rs:12 api/client.ts:236

Fix Right Away is a two-step flow:

  1. Diagnose — your description is sent to POST /api/diagnose. A short agent scans recently-modified files and returns a DiagnoseResult with root cause, affected files, and confidence score.
  2. Fix — once you confirm, a feature is created with the diagnosis as context and startRun is called immediately — the agent panel opens to show progress.

If diagnosis fails, the modal falls back gracefully — it skips diagnosis and creates the feature with your raw description.
