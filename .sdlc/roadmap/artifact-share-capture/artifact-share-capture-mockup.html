Specification: Artifact Share & Capture

Status: Approved · v1.2 · 2026-03-04

Overview

Users need one-click copy and capture on all major artifact surfaces. Currently sharing requires OS-level screenshot tools or manual text selection.

User Stories

  • → Copy raw markdown from any artifact in one click
  • → Copy rendered artifact as image for Slack / Discord
  • → Copy individual dialogue messages from ponder sessions

Technical Approach

Copy MD → navigator.clipboard.writeText(artifact.content)
Copy IMG → html2canvas(el) → ClipboardItem({ 'image/png': blob })
Download → html2canvas(el) → canvas.toDataURL() → anchor.click()

Click [MD] or [IMG] buttons above to see the copied feedback state. Hover buttons for tooltip labels.

Ponder session dialogue — hover over any message to reveal the copy action. Try hovering the messages below.
