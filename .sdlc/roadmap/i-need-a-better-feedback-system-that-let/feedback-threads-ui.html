Need a better feedback system for collaborating on ideas

open
opened by jordan · Mar 2 at 18:15 · 3 comments
Core element
v1 · original

The current feedback system (feedback.yaml) is a global capture queue — notes have no context, no author, and no persistence. When you hit "Submit to Ponder" it destroys the queue.

What's needed: a lightweight feedback thread where multiple people can contribute. The thread has a core element (a living summary that evolves) and a flat list of comments. An agent can synthesize comments back into the core element. When an idea is big enough, the thread promotes into a full Ponder workspace.
